Evangadi-forum Project

The project involved developing key components for the Evangadi Forum application, including the Header, Footer, Sign-up, Login, and About components, as well as the Sign-up and Sign-in pages and the Home page. Each component was designed with responsive styling and integrated with necessary APIs for user authentication and registration. The Header and Footer included essential navigation links and site information, while the Sign-up and Login components featured form validation to ensure data integrity. The About component provided insights into the Evangadi Network, and the Home page served as the user-friendly landing page for the application. This comprehensive implementation enhances the overall user experience and functionality of the forum.
